Detailed Description

Base_post is a Python base class, that inherits from the PostProcessor class.

It will house the most generic operations over all the post processors

Parameters

This is a place holder. Format similar to:

u: Base.Vector or WorkingPlane.PlaneBase, optional Defaults to Vector(1, 0, 0). If a WP is provided: A copy of the WP is created, all other parameters are then ignored. If a vector is provided: Unit vector for the u attribute (+X axis).
